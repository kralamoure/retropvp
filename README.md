# retropvp

`retropvp` is a game server for Dofus Retro.

## Requirements

- [Git](https://git-scm.com/)
- [Go](https://golang.org/)

## Build

```sh
git clone https://github.com/kralamoure/retropvp
cd retropvp
go build ./cmd/...
```

## Installation

```sh
go get -u -v github.com/kralamoure/retropvp/...
```

## Usage

```sh
retropvp --help
```
