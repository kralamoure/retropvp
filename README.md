# d1game

`d1game` is a game server for Dofus 1.

## Requirements

- [Git](https://git-scm.com/)
- [Go](https://golang.org/)

## Build

```sh
git clone https://github.com/kralamoure/d1game
cd d1game
go build ./cmd/...
```

## Installation

```sh
go get -u -v github.com/kralamoure/d1game/...
```

## Usage

```sh
d1game --help
```
